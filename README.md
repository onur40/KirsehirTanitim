# Kırşehir Tanıtım Web Sitesi

Bozkırın kalbi Kırşehir'in tarihi, kültürel ve gastronomik zenginliklerini tanıtan modern, mobil uyumlu ve yapay zeka destekli web projesi.

## Özellikler

- **Modern Tasarım:** Glassmorphism ve premium estetik.
- **Yapay Zeka Görsel Arama:** TensorFlow.js ve MobileNet kullanarak görsellerle yer arama.
- **Hızlı Filtreleme:** Kategori ve isim bazlı anlık arama.
- **Mobil Uyumlu:** Tüm cihazlarda kusursuz görünüm.

## Teknolojiler

- HTML5 & CSS3
- JavaScript (Vanilla)
- TensorFlow.js (AI Visual Search)
- FontAwesome & Google Fonts
